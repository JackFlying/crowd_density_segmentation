## data

This folder is where you put your own data. Please follow the rules:

1. Give a name (*e.g.* dataset_01) to the new dataset (*i.e.* collected by you or other public source).
2. For each dataset, it must have the following three subfolders:
    * **frames** contains frames extracted from videos.
    * **labels** contains labels made by label tools.
    * **perspective** conatins the perspective map for each scene.
